# jianshu_client
仿简书web页面

预览地址：https://jian.cemcoe.com/home/recommend

## 技术栈
vue2 + vuex + vue-router + webpack + ES6/7 + axios + flex


## 项目运行
需要先启动服务端

```
git clone git@github.com:cemcoe/jianshu_client.git

cd jianshu_client

npm install

npm run dev
```

## 目标功能
- [x] 首页推荐子页
- [ ] 文章详情页
- [ ] 个人页面
- [ ] 写文章页面
